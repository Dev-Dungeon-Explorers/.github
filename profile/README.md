### 개요

**던전탐험대**는 **Unity 6**를 사용하여 개발한 픽셀 아트 기반의 액션 디펜스 게임입니다.

</br>

던전 탐험대는 **유닛을 소환하여 적의 본거지를 무너뜨리면 승리**하는 간단한 디펜스 게임입니다.</br>
플레이어는 다양한 유닛들을 수집하고 이들의 조합을 갖추어 적들을 물리쳐야 합니다.

<br></br>

### 프로젝트 요약
